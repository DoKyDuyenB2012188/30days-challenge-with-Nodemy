# PRODUCT CART
## Tìm hiểu về
- object-fit: cover;
- object-position: center;
- transform: translateY(-90px) rotate(-20deg);
- opacity: 0;
- visibility: hidden;
- visibility: visible;
- outline: none;
- z-index: 2;
## Trick
- overflow: hidden trong body, nếu đặt ở cart thi mất ảnh
- opacity: 0  == visibility: hidden
- position nếu có thì không phụ thuộc vào padding, margin
